# Drent the hardware renting app
Email and Password Authentication In Flutter &amp; Firebase in Flutter 2.2

# Overview
This Drent the hardware renting app is implemented with many functionalities like, fluttertoast library, cloud firestore. In this tutorial, we have also implemented the Form validation for your email and password fields.
